            # Light bulb on/off with remote/timer

            # Introduction

            In this project, using KiMu's Automate Kit, we are going to build a system that uses a relay switch to automatically turn on/off the light bulb at specific time(s) or to turn on/off the light bulb using a remote control.

            2. Logical Thinking
